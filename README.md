## Overview
This project demonstrates hands-on experience building and managing a basic Microsoft Active Directory environment in a home lab. The purpose of this project is to practice real-world identity and access management tasks commonly performed in enterprise IT and managed service provider (MSP) environments.

## What I Built
- Deployed a Windows Server domain controller and created a new Active Directory domain (corp.local)
- Installed and configured Active Directory Domain Services (AD DS)
- Created and managed domain user accounts (new hires, password resets, account lockout/unlock)
- Configured account lockout policy using Group Policy and validated behavior
- Joined a Windows 10 workstation to the domain and verified domain authentication

## Technologies Used
- Windows Server 2019/2022
- Windows 10
- Active Directory Domain Services (AD DS)
- Group Policy Management
- VirtualBox

## Skills Demonstrated
- User lifecycle management
- Password policy and account lockout configuration
- Windows domain troubleshooting
- Workstation deployment and domain joining
- Documentation and validation of technical work

## Why This Project Matters
This project reflects real-world tasks performed by IT Support, Deployment, and Junior Systems Administrator roles, including user management, workstation onboarding, and troubleshooting authentication issues.
